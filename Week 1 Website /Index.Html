Let''s Get Started

This is the beginning of week one in our class, and we are starting our journey into web development. We are learning how to code using HTML and CSS, which are the fundamental building blocks of websites. Our main goal is to apply these skills by creating a personal project website. This project will help us practice structuring content with HTML and styling it with CSS, laying the foundation for more advanced coding in the weeks to come.

Building in VSCode

Visual Studio Code (VSCode) allows you to write, experiment with, and test your code efficiently. By using extensions or built-in tools like Live Server, you can run your website files on a local server. This lets you preview changes in real-time as you build and refine your project, making the development process faster and more interactive.
